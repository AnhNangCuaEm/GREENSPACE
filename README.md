# GreenSpace ウェブサイト

GreenSpaceは、都市生活の忙しさや慌ただしさの中でリラックスできる場所を提供することを目的とした、公園やイベント情報を発信するウェブサイトです。現代的で鮮やかな色彩とブラー効果を取り入れたデザインが特徴です。

---

## 主な機能

- ユーザーログイン・ログアウト
- リアルタイム検索と即時結果表示
- プロフィールカスタマイズ（アバター変更・情報追加）
- データはすべてデータベースに保存
- その他

---

## 管理者専用機能
- ダッシュボード（アクセス数、保存された数、コメント、などの表示）
- ユーザー管理
- 公園・イベント情報の管理（追加・削除・編集）
- 通知の作成・送信
- フィードバック管理
